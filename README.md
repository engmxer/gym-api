# Gym API

API для управления тренажерным залом. Позволяет управлять клиентами, тренировками и записями на тренировки.

## Функционал

- Управление клиентами (CRUD операции)
- Управление тренировками (CRUD операции)
- Управление записями на тренировки (CRUD операции)
- Фильтрация списков по различным параметрам
- Массовое создание, обновление и удаление записей
- Автоматическая документация Swagger UI
- Админ-панель Django

## Технологии

- Python 3.x
- Django 5.x
- Django REST Framework (DRF)
- drf-spectacular (документация)
- SQLite (база данных)
