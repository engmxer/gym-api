# Gym API

**API для управления тренажерным залом**  
Разработано на Django REST Framework (DRF)

## О проекте

Gym API предоставляет бэкенд для системы управления тренажерным залом. Позволяет:
- Управлять клиентами (добавление, редактирование, удаление)
- Управлять тренировками (виды занятий, цены, длительность)
- Управлять записями клиентов на тренировки
- Фильтровать данные по различным параметрам
- Выполнять массовые операции (создание, обновление, удаление нескольких записей)

## Архитектура проекта

Проект построен на архитектуре REST API с использованием:
- **Django** - веб-фреймворк
- **Django REST Framework** - создание API
- **SQLite** / **Microsoft SQL Server** - база данных (поддерживаются обе)
- **drf-spectacular** - автоматическая документация Swagger
